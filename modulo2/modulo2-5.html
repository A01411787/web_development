Oportunidades y Amenazas

Al terminar el Análisis de Atractividad de la Industria, el siguiente paso consiste en realizar la primera parte de un análisis llamado FODA.

Este nombre está compuesto por las siglas que significan Fuerzas, Oportunidades, Debilidades y Amenazas. Pero para fines de este análisis, sólo revisaremos las Oportunidades y las Amenazas que presenta la industria para tu negocio. De esta forma concluye el análisis de los factores externos que influyen en el negocio.

Para realizar este análisis se requiere que utilices dos tablas de evaluación. Una para las oportunidades y otra para las amenazas. A continuación encontrarás la forma en cómo debes llevar el análisis para cada uno de estos aspectos.
