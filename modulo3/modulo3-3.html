Proceso para determinar las bases de competencia

A continuación presentamos un proceso que te ayudará a determinar las bases de competencia para tu negocio. Los resultados de este análisis se utilizarán para decidir estrategias y tácticas por lo tanto, es la clave para la realización de cualquier plan de mercado. Da un clic en cada etapa del proceso.

Haz clic en los tipos de usos para saber más.

Establecer hipótesis

En este primer paso, se establecerá una hipótesis o suposición de las bases de competencia de tu negocio. Hay que tomarla a criterio, considerando las bases de competencia más relevantes, de acuerdo al producto y/o servicio. Ejemplo Recuerda que para varias bases de competencia lo importante es la percepción de los clientes; por ejemplo la "calidad" de tu producto y/o servicio.

Calificar las bases de competencia

Al tener identificadas las bases de competencia, es necesario ordenar cada una de ellas otorgándoles una calificación. Es recomendable explicar el porqué de la evaluación. En este análisis se puede realizar la evaluación de acuerdo a las empresas de la competencia o de productos sustitutos que se encuentran en la industria; ya con este diagnóstico se puede extrapolarlo a las características desarrolladas para tu negocio.

Priorizar de acuerdo a la calificación

En este paso, se debe priorizar, a partir de la calificación otorgada a la hipótesis o suposición de cada una de las bases de competencia que se haya seleccionado a la empresa.

Validar las bases de competencia

Valida cada una de las bases de competencia que hayas supuesto o seleccionado. Forma de validar las bases de competencia Para llevar a cabo la validación respectiva, debes conocer cómo manejan las mismas bases de competencia los competidores más fuertes. Esto lo puedes saber por investigación directa (ejemplo: por observación). Otra manera puede ser al contemplar la validación dentro del diseño que se hará posteriormente para la investigación del mercado meta hayas definido. Se puede llevar a cabo la validación con la cadena de distribución y con los proveedores. Pero esta forma, es recomendable, cuando ya se tenga algún tiempo operando y deseamos volver a hacer este análisis.

Determinar la posición competitiva

Con base al análisis anterior de las bases de competencia y la validación de las mismas, podremos determinar las posiciones competitivas correspondientes.
